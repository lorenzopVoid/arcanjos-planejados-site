# Arcanjos Planejados

Landing page institucional desenvolvida para a **Arcanjos Planejados**, empresa especializada em móveis planejados sob medida para ambientes residenciais e corporativos.

O projeto foi criado com foco em uma experiência premium, design moderno, alta performance e geração de leads para captação de novos clientes.

---

## ✨ Demonstração

**Site:** Em breve

**Deploy:** Vercel

---

## 📋 Sobre o Projeto

A plataforma foi desenvolvida para apresentar os serviços da Arcanjos Planejados de forma profissional, transmitindo credibilidade, sofisticação e facilitando o contato entre a empresa e seus clientes.

O site possui navegação responsiva, galerias organizadas por ambiente, animações suaves e formulário de orçamento integrado ao WhatsApp e e-mail.

## 🖼️ Portfólio

As galerias são organizadas por ambiente e carregadas automaticamente através da estrutura de pastas do projeto.

## ▶️ Executando Localmente

Clone o repositório:

```bash
git clone https://github.com/lorenzopVoid/arcanjos-planejados-site
```

Acesse a pasta:

```bash
cd arcanjos-planejados
```

Instale as dependências:

```bash
npm install
```

Execute o projeto:

```bash
npm run dev
```

Acesse:

```txt
http://localhost:5173
```

---

## 📦 Build para Produção

Gerar build:

```bash
npm run build
```

Visualizar build:

```bash
npm run preview
```
